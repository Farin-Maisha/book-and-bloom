##  Project Overview

###  Project Title
**Book & Bloom – AI-Powered Digital Library System**

###  Objective
The objective of this project is to build a smart and user-friendly digital library platform that simplifies book searching, reservation, and tracking while providing intelligent assistance through an AI chatbot. The system improves organization, reduces manual workload, and enhances user experience for readers and library administrators.

### Target Audience
- Students and academic readers  
- Educational institutions and libraries  
- Book lovers and cozy readers  
- Small libraries seeking digital management solutions  

### Project Type
- AI-Augmented Learning & Education Platform  
- Productivity System  
- Small Business & Educational Support System

## Project Features

### Authentication
- User Registration & Login  
- JWT-based authentication (planned)

###  Book Management
- View all books
- Search books by title, author, category
- View availability status
- Reserve books

###  Intelligent Chatbot Assistant (Exclusive Feature)
- AI-powered chatbot helps users:
  - Find books instantly
  - Answer library related queries
  - Guide users through system navigation
  - Provide recommendations (future scope)
- Improves decision support and usability.

###  MyLibrary Dashboard
- View issued books
- Track due dates
- Monitor borrowing history

###  Category Browsing
- Organized book filtering by categories
